### 用法

| 自定义属性                   | 描述               |
| ---------------------------- | ------------------ |
| dora_clearIcon               | 右侧清空的按钮     |
| dora_clearIconVerticalMargin | 清空按钮垂直的边距 |
| dora_clearIconMarginLeft     | 清空按钮的左边距   |
| dora_clearIconMarginRight    | 清空按钮的右边距   |
